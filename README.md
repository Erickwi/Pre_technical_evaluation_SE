# SE Vuelo

## Prerequisites

- Install and configure Git
- Install Java 11 and config JAVA_HOME environment variable
- node.js and npm
- Install globally Angular CLI version 10

## Execute in DEV environment

### 1. Backend
```
cd sevuelo-backend
./mvnw spring-boot:run
```


### 2. Frontend
```
cd sevuelo-frontend
npm install
ng serve --open
```
